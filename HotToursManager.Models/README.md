HotToursManager.Models
📦 Библиотека моделей для приложения HotToursManager.
📖 Описание
Данный пакет содержит основные сущности предметной области, используемые в приложении для управления турами.
Включает:

    модели данных (например, Tour, Statistics)
    атрибуты валидации (через TourLimits)
    базовые структуры для передачи данных между слоями

Пакет предназначен для повторного использования в различных слоях приложения:

    сервисы (Business Logic)
    пользовательский интерфейс (WinForms)
    другие возможные клиенты

🚀 Установка

1

или через Visual Studio:

    Установить

🧩 Пример использования

csharp
using HotToursManager.Models;

var tour = new Tour
{
    Destination = "Турция",
    Nights = 7,
    DepartureDate = DateTime.Now,
    CostPerPerson = 50000,
    NumberOfPeople = 2,
    HasWiFi = true,
    Surcharges = 5000
};

Console.WriteLine($"Общая стоимость: {tour.TotalCost:N2} ₽");
// → Общая стоимость: 735000,00 ₽

📌 Назначение
Пакет реализует слой контрактов и не содержит:

    бизнес-логики
    пользовательского интерфейса

Это позволяет:

    переиспользовать модели в разных проектах
    уменьшить связанность между слоями
    упростить сопровождение системы
